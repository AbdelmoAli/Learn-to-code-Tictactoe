Let's Begin!

The Python language uses different data types, that uses different operations. The first type is the Integer type. If you're not bad in mathematics,you should know about the common operations you can do with integers: addition (+), substraction (-), multipl- ication (*), and division (-).

Well, in Python you can use those different operations with the integers. But you can do a lot more! If you want to learn more ab- out it, check a documentation on the internet. For now, that is more than enough to continue our lesson.

The second data type that we'll talk about is the String type. A String is just a succession of characters that forms a word, a sentence, a paragraph or a whole book. Exemples: "a", "b", "word", "this is a sentence" are strings. As you notice, strings are always written between "", which enables Python to understand that you wrote a string.

The last data type you'll need to know about is the List type. A list is a collection of objects that is orderer and changeable. For instance, [1] is a list containing the integer 1, [1,2,3] is a list containing the first three integers. You can also have an empty list [].

How to declare a variable?

A variable allows you to store a certain data under a name of your choice. Here's an example: 

my_variable =  '1'

Here you stored the value 1 under the name my_variable. What you do in general respect the following structure: 

name_of_variable = ' value'

Value can take of all sorts of data types : integer, string, list and so forth.

What about functions?

A function takes a bunch of parameters (that can be Integers, Strings, Lists,..), does some actions and return a new value. To write a function means to define what sort of parameters it takes ( their number and their type), then define the actions that are applied to them and then return the results of the actions.

def function(parameters_1,parameters_2):
    ---some actions ---
    ..
    ..
    return output_1, output_2
>

For instance, here's how you would define a function that takes two integers and returns their sum: 

def sum(a,b):
    c=a+b
    return c

Here we took the parameters a and b and we did the operation a+b which we stored the result in c (second line) then we returned c. Lastly, you need to know that you can define functions that does not need to take a parameter. For instance here's a function that returns the number 1 no matter what : 

def return_one(1):
    return print(1)

Indentation

When you write python code, you need to organize it in a way such that python can understand it. For now, you'll need to know that whenever you use ":" statement you need to write in the next line and start your code with leaving some space. The spaece you leave is up to you but you need to keep leaving the same amount of space after each ":" statement. For instance: 

def function():
    a=4
    if a==3:
       a=4
    return a

You can notice that after the first ":" statement i left 4 empty spaces and i respect that amount each time i wrote that statement once i'm done with a ":" statement and it's content, i go back to write with the same amount of space i used in the previous ":" line. For instance, the following code is wrong: 

def function():
    a=3
    if a==3:
       a=4
    return 4

For loops

For loop is the way python allows you to repeat a certain block of operations as many time as you want. Bear in mind that in python we count strating from 0. So here's how you can repeat the operation (print(1)) 10 times: 

def function():
    a==3
    if a==3:
    a=4
    return a

you'll get the result : 1111111111. You need to know that range(10) means that you can access to the numbers between 0 and 9. In general to repeat a certain operation n times you just write: 

for i in range(10):
    operation(i)

For instance here's a very easy way to fill a list when you want to fill it with the result of a fucntion. Suppose you have a function 'operation' that takes an integers and operates on it. If you want to create a list that contains the results of applying this function to each of the integers between 0 and 10 you will just write: 

name_of_list = [operation(i) for i in range(11)]
